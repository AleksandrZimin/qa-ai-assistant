# Кэш контекста сессий

Сюда агент сохраняет краткие итоги сессий (навык `save-context`): что делали,
какие решения приняли, что осталось незакрытым. В новой сессии агент читает
эти файлы и продолжает работу без пересказа.

- **Срок хранения — 2 дня.** Более старые файлы удаляются автоматически при
  старте сессии (хук `.claude/scripts/clean-cache.ps1`).
- Папка личная для каждого тестировщика и не попадает в git.
- Нужно сохранить итог надолго — попросите агента положить его в `results/`.
