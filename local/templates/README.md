# local/templates/ — адаптированные шаблоны вашей команды

Если ваша команда ведёт тест-кейсы или отчёты по требованиям в отличном от общего
формате — положите сюда файл с **тем же именем**, что в `templates/`
(`test-case-template.md`, `test-case-example.md`, `Otchet-o-testirovanii-trebovanij.md`),
и агент будет использовать его вместо общего.

Если файла здесь нет — используется общий шаблон компании из `templates/`.
